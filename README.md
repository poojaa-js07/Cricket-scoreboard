# 🏏 Professional Cricket Scoreboard

A responsive cricket scoreboard built using HTML, CSS, and JavaScript.

## Features

- Team Selection
- Live Score Updates
- Overs Calculation
- Current Run Rate (CRR)
- Wide & No Ball
- Wicket Counter
- Switch Innings
- Target Calculation
- Winner Display
- Match Reset
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Clone the repository.
2. Open the project in VS Code.
3. Open `index.html` using Live Server.
